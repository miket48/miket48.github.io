Instructions

Hello and welcome to Pictionary!

Below is all you need to get started:

Checklist:
  1. An internet connection. [ Check ;) ]
  2. Two internet connected devices.
    • A viewing device. (Preferably big screen e.g. Laptop) [ ]
    • A controller device. (Needs to have some sensors e.g. Smartphone) [ ]
  3. Some mad drawing skills. [ jk I can't draw either :( ]

Login/Sign-up:
  1. Click on the "Login/Signup" button on the top right of the screen.
  2. Click on "Login" if you already have an account.
  3. If you are new, no worries, just click the "Sign-up" button.
  4. Which ever you selected enter all the relevant details.
  5. Click on proceed and now you are ready to join or create a game :D!

To Join an Existing Game:
  1. Firstly Login/Sign-up. (See above for details)
  2. Ask your friend/family/lover/drinking partner for the Server ID displaying on their screen (when they have created a room).
  3. Paste the Server ID into the Server ID field next to the "Join Room" button.
  4. Click on "Join Room".
  5. Have fun!

To Join your Controller:
  1. Firstly Login/Sign-up. (See above for details)
  2. Ask your friend/family/lover/drinking partner for the Server ID displaying on their screen (when they have created a room).
  3. Paste the Server ID into the Server ID field next to the "Connect Device" button.
  4. Click on "Connect Device".
  5. Good luck!

To Create a Game:
  1. Firstly Login/Sign-up. (See above for details)
  2. Click on "Create Room".
  3. It is that easy! xD
  4. (Remember to send the Server ID to your subordinates)
